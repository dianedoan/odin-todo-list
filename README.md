# Odin Todo List
This [webpage](https://dianedoan.github.io/odin-todo-list/) is a basic Todo List application.

The user interface includes the following features:
- View all projects
- View all todos in each project (probably just the title and duedate… perhaps changing color for different priorities)
- Expand a single todo to see/edit its details
- Delete a todo

This app uses the Web Storage API `localStorage` which allows users to save their data on their computer. 

## Description
This project demonstrates the skills learnt from the JavaScript Course in The Odin Project up until the [OOP Principles](https://www.theodinproject.com/lessons/node-path-javascript-oop-principles) topic.
